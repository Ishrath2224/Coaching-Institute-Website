# Coaching Institute Website

A modern and responsive website designed for coaching institutes and educational organisations. The platform provides students and parents with easy access to course information, faculty details, success stories, and admission procedures through a clean and user-friendly interface.

## Features

* Responsive design for desktop, tablet, and mobile devices
* Course and program showcase
* Faculty and mentor profiles
* Student testimonials and success stories
* Admission inquiry/contact form
* Modern and intuitive user interface

## Tech Stack

* HTML
* CSS
* JavaScript
* React

## Objective

The goal of this project was to create a professional online presence for educational institutions, enabling them to showcase their offerings and simplify the student enrollment process.
